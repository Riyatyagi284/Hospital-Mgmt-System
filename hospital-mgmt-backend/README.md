## Features

- RESTful API design using proper HTTP methods and status codes

- Pagination support with pagination links for better API discoverability

- Consistent API response structure

- Input sanitization and request validation

- Structured logging implementation

- Detailed error logging for easier debugging

- search functionality across multiple fields

- Soft delete and hard delete support

- filter support

- Scalable and maintainable folder structure

- TypeScript support

- Clean architecture principles:

  - No business logic inside controllers

  - No direct database calls from controllers

  - Service layer architecture implementation

- Unit test cases added
