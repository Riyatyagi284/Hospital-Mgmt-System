This is my Avdar's Assignment Frontend .

## Frontend Features

- Built reusable and maintainable UI components
- Styled using Tailwind CSS
- TypeScript support for better type safety and maintainability
- Axios integration for API communication
- Centralized API client architecture
- No direct API calls inside components
- TanStack Query (React Query) used for efficient server state management
- Zustand used for lightweight UI/global state management
- Proper error handling implementation
- Loading and error state handling across the application
- Clean and scalable frontend folder structure
